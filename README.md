## 🚀 快速开始

### 一键安装（推荐）
```bash
curl -sSL https://raw.githubusercontent.com/Fanxing-9049/MMA_Pro/main/install.sh | bash
```

### Docker部署
```bash
# 下载配置文件
curl -o docker-compose.yml https://raw.githubusercontent.com/Fanxing-9049/MMA_Pro/main/docker-compose.yml

# 启动服务
docker-compose up -d

# 访问应用
open http://localhost:8500
```

### 手动部署
```bash
# 克隆仓库
git clone https://github.com/Fanxing-9049/MMA_Pro.git
cd MMA_Pro

# 启动服务
docker-compose up -d
```

## 🏗️ 技术架构

### 支持平台
- ✅ **Linux x86_64** - Intel/AMD 64位处理器
- ✅ **Linux ARM64** - ARM 64位处理器（树莓派4、Apple M1等）
- ✅ **Docker** - 跨平台容器化部署
- ✅ **云服务器** - 支持各大云服务商

### 系统要求
- **操作系统**: Linux (Ubuntu 18.04+, CentOS 7+, Debian 9+)
- **内存**: 最少2GB RAM，推荐4GB+
- **存储**: 最少1GB可用空间
- **网络**: 需要互联网连接
- **Docker**: 推荐使用Docker部署
